Tags H1 até H6

As tags de cabeçalho 'h1' até 'h6' no HTML são usadas para definir cabeçalhos em uma página web, onde 'h1' é o mais importante e 'h6' o menos importante. Aqui está uma breve descrição de cada uma:

  1. h1: Usado para o título mais importante ou principal. Geralmente cada página deve ter apenas um h1 para manter a hierarquia semântica e para fins de SEO.
  2. h2: Usado para sub-títulos ou seções importantes. É o segundo nível de cabeçalho e deve ser usado pçara dividir o conteúdo em seções menores.
  3. h3: Usado para sub-sub-títulos ou subseções dentro de uma seção h2.
  4. h4: Usado para subdivisões ainda menores dentro de uma seção h3.
  5. h5: Usado para subdivisões dentro de uma seção h4.
  6. h6: Usado para o nível mais baixo de cabeçalho, subdividindo seções h5.

Cada nível de cabeçalho deve ser usado de forma hierárquica e organizada para melhorar a acessibilidade e a SEO da página. (voltar ao indice)

Tags P, MARK, I, U, STRONG

A tag 'P' é usada para definir um parágrafo. O navegador adiciona automaticamente uma linha em branco antes e depois de cada parágrafo.

A tag 'MARK' é usada para destacar o texto que é relevante ou importante. Normalmente, o texto dentro da tag 'MARK' é destacado com um fundo amarelo.

A tag 'I' é usada para definir um texto em itálico. Esta tag pode ser usadapara indicar uma ênfase em uma palavra ou frase.

A tag 'U' é usada para sublinhar o texto. Pode ser útil para enfatizar palavras ou frases específicas.

A tag 'STRONG' é usada para definir texto importante e geralment é renderizada em negrito pelos navegadores. Além da ênfase visual, ela também indica uma ênfase semântica mais forte.

Essas tags ajudam a estruturar e estilizar o conteúdo de uma página web ded forma que ele fique mais legível e organizado.(voltar ao indice)

Tags OL, UL, li

As tags 'OL' 'UL' e 'LI' são usadas para criar listas em HTML. Aqui vai uma breve descrição de cada uma:

  1. 'OL' (Ordered List): é usada para criar uma lista ordenada, onde os itens são numerados automáticamente pelo navegador.
  2. 'UL' (Unordered List): é usada para criar uma lista não ordenada, onde os itens são marcados com marcadores (normalmente pontos).
  3. 'LI' (List Item): é usada para definir um item em uma lista, seja ela ordenada ('OL') ou não ordenada ('UL').
